Display: Inline, Inline-Block or Block?

The CSS display property has several different values, which all interact in a different way with the contents of an element. Before we start, let's look at how we got here.

We start out with an element in HTML - for this example, I'll use a <span> (entirely because that's close to spaniel, and I wanted to make that pun). Seeing as a <span>(iel) is a container for just a small amount of content, it is probably nested inside of some <p> tags. So here's my <span> within a paragraph, and naturally, I've given it a class of spaniel. Within CSS, I can now give this spaniel class a display property.

display: inline;
